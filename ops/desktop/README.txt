ATLAS OPERAZIONI — Guida rapida
================================

1. Esegui UNA VOLTA: installa-sul-desktop.bat
   (copia questa cartella sul Desktop)

2. Ogni giorno apri: APRI-QUESTO.html

3. Ogni 2 giorni:
   - Prompt AI (vedi docs/operativo/PROMPT_RICERCA_EVENTI.md)
   - Control Center → Scoperta → incolla → Pubblica
   - Opzionale: Aggiorna-Fonti.bat

4. Segnalazioni utenti:
   - Control Center → Segnalazioni
   - Codice riferimento ATL-... per tracciabilità WhatsApp

5. Correggere un evento sulla mappa:
   - Control Center → Mappa → clic pin → Modifica → Salva

Per sviluppo e modifiche al codice: usa la chat Cloud Agent, non questa cartella.
